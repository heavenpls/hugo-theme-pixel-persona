# Pixel Persona v1.0.0

首次公开发布。

Pixel Persona 是一款面向个人博客与技术写作者的 Hugo 主题，强调像素风视觉、长文阅读体验、深色代码块与内容优先的信息结构。

本次 `v1.0.0` 为首个可公开使用版本，已具备完整的主题基础能力，可用于个人博客、开发者知识库、技术随笔站点与长期写作场景。

## 本次发布内容

### 主题基础能力

- 提供独立的 Hugo 主题结构
- 支持首页、文章页、标签页、分类页、404 页
- 支持亮色 / 暗色主题切换
- 支持响应式布局
- 支持中文博客场景

### 首页体验

- 像素风 Hero 区块
- 状态面板展示
- 最新文章网格
- 归档时间轴入口

### 文章页体验

- 更宽的长文阅读舞台
- 右侧文章信息栏
- 目录侧栏
- AI 摘要区块
- 更适合技术文章的正文排版

### 标签与内容导航

- 标签总览卡片云
- 标签详情时间轴布局
- 内容优先的信息结构

### 代码块增强

- 代码块顶部工具栏
- 语言标识
- 一键复制按钮
- 复制成功反馈
- 自定义深色语法高亮配色
- 长代码块在正文列内安全滚动，不破坏页面布局

## 设计特点

- 不是纯游戏化像素界面，而是更适合内容站的复古像素风
- 保留较强辨识度，同时兼顾长文阅读体验
- 适合技术博客、运维文档、架构文章与开发记录

## 适用场景

- 个人博客
- 技术知识库
- 开发者作品与文章站
- 运维 / 架构 / 编程随笔
- 长篇技术文章与排障复盘

## 安装方式

### Git Submodule

```bash
git submodule add git@github.com:heavenpls/hugo-theme-pixel-persona.git themes/pixel-persona
```

### Git Clone

```bash
git clone git@github.com:heavenpls/hugo-theme-pixel-persona.git themes/pixel-persona
```

## 使用要求

- Hugo `>= 0.160.0`
- 建议启用：

```toml
[markup.highlight]
  noClasses = false
```
